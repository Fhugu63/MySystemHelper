# About
This is a project written in C++, The idea is that it should help me with things on my computer.

# What I expect from the project
I expect that this assistant will help me with some of the things on my PC, for example, I plan to sort files from downloads, make some applications launch based on it, in theory, This can protect your PC from viruses. Also, if you looked at the code, you may have noticed that I'm adding the ability to count examples by entering them in full in any format. In theory, it could also include PC performance analysis and monitoring of suspicious applications.

I don't know, but it seems to me that this is a good option since you can create functionality of varying complexity.
